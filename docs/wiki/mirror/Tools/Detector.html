Detector Tools

dets_extractValues.py

Reads a list of detector positions using the DFROUTER detector positions format (XML) from the first given file. Then parses the second given file which is assumed to have detector values stored in the DFROUTER detector value format and writes only those detectors into the third given file which were found in the first file.

Call: 

dets_extractValues <DETECTOR_POSITIONS> <INPUT_VALUES> <OUTPUT_VALUES>
