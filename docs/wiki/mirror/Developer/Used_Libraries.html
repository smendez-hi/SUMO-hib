Developer/Used Libraries

Included in the SUMO-source - package

  • nvwa: for memory checks
  • Mersenne Twister Random Number Generator: for generating random numbers
  • Polyfonts for text drawing in openGL
  • RTree library by Greg Douglas for somehow fast visualization
  • eulerspiral library by Amir Tamrakar for clothoid computation (curently only within open drive import)
  • gl2ps for exporting vector images

To be downloaded separately

  • xerces-c: for XML-parsing; versions 3.0.x work fine
  • FOX-toolkit: for windowing (GUI); version 1.6.x is needed
  • proj.4 for projecting geo-coordinates; version 4.x should work
  • gdal for shapefile-import; version 1.3.1 works, had problems with version 1.3.2, recent 1.5.x versions work
